#Geobeans 项目复用框架

为了方便项目的快速搭建，整理并编写了基于java的后端框架blankServer,和前端框架blank。
在实际的项目中，可以clone该项目进行快速开发。

具体的git 命令：

> git clone https://git.oschina.net/ghx/blank.git
